One-time setup required
To connect this app to your Google Sheet so all bookings are shared in real time, you need to set up a Google Apps Script. This takes about 5 minutes and only needs to be done once.
1
Open your Google Sheet: Click here to open it
2
Click Extensions in the top menu → then click Apps Script
3
Delete everything in the editor and paste the script below, then click the Save (💾) icon
4
Click Deploy → New deployment → set type to Web app → set "Who has access" to Anyone → click Deploy → copy the Web App URL
5
Paste the Web App URL into the box below and click Save
